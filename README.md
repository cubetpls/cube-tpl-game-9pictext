# 标准游戏 cube 模板，9图文内容

## 介绍
标准游戏类模板，适合游戏类场景

## 使用
```
cubetool init <cubeid> -t https://github.com/ntt360/cube-tpl-game-9pictext.git
```
或简写：
```
cubetool init <cubeid> -t ntt360/cube-tpl-game-9pictext
```
